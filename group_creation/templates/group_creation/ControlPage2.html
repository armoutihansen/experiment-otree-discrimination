3. Lotterien: Aus Ihrem Alltag ist Ihnen die Funktionsweise von Lotterien wahrscheinlich schon bekannt. Dennoch wird es hilfreich sein Sie zunächst damit vertraut zu machen, wie Lotterien in diesem Experiment durchgeführt werden.

In einer Lotterie werden die Ergebnisse durch Zufall bestimmt. Betrachten Sie zum Beispiel eine Lotterie bei der zwei Auszahlungen, 100 Punkte und 20 Punkte, möglich sind. Es besteht eine Wahrscheinlichkeit von 60%, dass das Ergebnis 100 Punkte beträgt und eine Wahrscheinlichkeit von 40%, dass es 20 Punkte beträgt.

Eine Möglichkeit sich diese Lotterie vorzustellen ist die folgende. Nehmen Sie an es gibt eine Urne mit 100 Kugeln von denen 60 rot und 40 blau gefärbt sind. Eine Kugel wird zufällig ohne hinzuschauen aus der Urne gezogen. Falls die Kugel blau ist gewinnen Sie 100 Punkte, bei einer roten Kugel gewinnen Sie 20 Punkte. Wie Sie nachvollziehen können sollten ist die Wahrscheinlichkeit, dass eine zufällig gezogene Kugel blau ist exakt 60% und dass sie rot ist exakt 40%.
